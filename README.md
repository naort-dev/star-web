# Web App for Starsona

## Getting Started
```
# Clone repo
git clone git@github.com:Starsona/web.git

# Navigate to repo
cd web

# Install deps
npm install

# To run the project 
npm start

open http://localhost:8080 in the browser to see the running application.  

# For Creating a build 
npm run build , build will be created in the dist folder.

```
